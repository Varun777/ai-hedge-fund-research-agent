# AI Hedge Fund Research Agent

An AI-powered equity research system that scans financial news daily, resolves public companies against an official security master, enriches them with market data, ranks opportunities using dynamic signal scoring, and generates hedge-fund-style investment memos.

## Features

- Financial news ingestion
- AI company extraction
- Security master ticker resolution
- Market reaction analysis
- Dynamic signal scoring
- Liquidity/tradability scoring
- Earnings timing classification
- Entry-type classification
- Investment thesis generation
- Claim discipline (facts vs inference vs hypothesis)
- Daily memo generation

## Stack

- Python
- OpenAI API
- yfinance
- RapidFuzz
- RSS feeds
- GitHub Actions (planned)

## Example Workflow

1. Pull daily market news
2. Extract public companies with AI
3. Match companies against listed securities
4. Fetch market and liquidity data
5. Score opportunities
6. Generate investment committee memo
